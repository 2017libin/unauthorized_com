# unauthorized_com
# 说明
未授权检测的命令行版V1.0，支持批量检测，导出结果，项目参考sqlmap使用随机user-agent。本项目有两个版本，一个是带有GUI界面的，一个是命令行版本，未授权检测目前不包括默认密码检测
